kamuran
=======

Random suggestion helper, unnecessary at all.

##Install

```bash
	$ npm install -g kamuran
```

##Usage

```bash
 $ kamuran nereye gidelim?
```

There are a few alias parameters for suggestion. They are:
```
[
	'suggest'
	'nereye gidelim'
	'nereye gidelim?'
	'nereye gideyim'
	'nereye gideyim?'
	'ne dersin'
	'ne dersin?'
]
```
